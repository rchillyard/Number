Angle

com.phasmidsoftware.number.algebra.eager.Angle
See theAngle companion class
object Angle

The Angle companion object contains utility methods, predefined constants, and typeclass instances for working with angles. Angles are represented using rational numbers and comply with the algebraic structure of a commutative group.
