def toLong(x: Number): Long

Converts the given Number instance to a Long representation. If the number is a GeneralNumber and contains a rational value, the rational is converted to a Long. If the rational is absent but a double value is present, it rounds the double to the closest Long. Throws an exception if neither a rational nor a valid double representation can be derived.
