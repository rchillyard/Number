implicit class FuzzStringOps(w: String)

Implicit class which takes a String, and using method ~ and an Int parameter, yields a Number with the appropriate degree of fuzziness. This class provides an alternative to having to parse a fuzzy number from a String. Really, the only function it provides is the ability to put the error bounds after the exponent. It is parallel to FuzzOps except for two differences: the input is a String, and this is is so that trailing zeros in the fractional part don't get ignored, thus messing up the fuzziness; And, secondly, the result of ~ is a Try[Number], not just a Number.

final implicit def FuzzStringOps(w: String): FuzzStringOps

Implicit class which takes a String, and using method ~ and an Int parameter, yields a Number with the appropriate degree of fuzziness. This class provides an alternative to having to parse a fuzzy number from a String. Really, the only function it provides is the ability to put the error bounds after the exponent. It is parallel to FuzzOps except for two differences: the input is a String, and this is is so that trailing zeros in the fractional part don't get ignored, thus messing up the fuzziness; And, secondly, the result of ~ is a Try[Number], not just a Number.
