RomanParser

com.phasmidsoftware.number.top.RomanParser
class RomanParser extends JavaTokenParsers

The RomanParser class provides functionality for parsing Roman numerals and converting them into their respective natural number representations using the Natural type system.

The class uses combinator parsers to validate and parse the Roman numeral strings. Supported Roman numeral characters are M, D, C, L, X, V, and I. The parser respects the rules of Roman numeral construction, including subtractive combinations like IV and IX.

case class ~[+a, +b](_1: a, _2: b)

A wrapper over sequence of matches.

A wrapper over sequence of matches.

Given p1: Parser[A] and p2: Parser[B], a parser composed with p1 ~ p2 will have type Parser[~[A, B]]. The successful result of the parser can be extracted from this case class.

It also enables pattern matching, so something like this is possible:

def concat(p1: Parser[String], p2: Parser[String]): Parser[String] =
  p1 ~ p2 ^^ { case a ~ b => a + b }
