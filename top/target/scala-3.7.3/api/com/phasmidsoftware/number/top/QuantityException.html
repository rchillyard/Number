com.phasmidsoftware.number.top.QuantityException
case class QuantityException(message: String) extends Exception

Represents an exception specific to quantity-related errors.

The QuantityException is used to signal issues or invalid operations related to quantities within the application. It extends the base Exception class and includes a customizable error message to provide additional context about the exception's cause.

message

Detailed message describing the nature of the quantity-related error.
