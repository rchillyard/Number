ExpressionParser

com.phasmidsoftware.number.parse.ExpressionParser
object ExpressionParser

The ExpressionParser object provides utilities for parsing and processing LaTeX-like mathematical expressions, leveraging string interpolation. These utilities enable creation, simplification, and evaluation of mathematical expressions represented as Expression objects.
