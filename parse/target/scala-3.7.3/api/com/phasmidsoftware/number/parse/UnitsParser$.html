UnitsParser

com.phasmidsoftware.number.parse.UnitsParser
object UnitsParser

Object responsible for parsing unit expressions into their corresponding Unit representations. This supports basic SI units, common derivations, unit powers, products, and divisions.

def parse(input: String): Either[ParseError, Unit[_]]

Parses the given input string into a unit of measurement or returns an error message.

Parses the given input string into a unit of measurement or returns an error message.

The method attempts to parse a unit expression from the input string using the defined unitsParser. If the parsing is successful, it returns a Right containing the parsed unit. If the parsing fails, it returns a Left containing a detailed error message.

Value parameters

input

the input string to parse, representing a unit expression

Attributes

Returns

an Either where Right contains the parsed unit or Left contains the error message NOTE that because of the way the parse method works, we can't easily distinguish between parse errors and unit errors.

def unitsParser(using ParsingRun[Any]): ParsingRun[Unit[_]]

Parses a complete unit expression involving division.

Parses a complete unit expression involving division.

This method constructs a parser that recognizes and processes unit expressions combining products of units (e.g., "mÂ·s" or "kg*m") with optional division operators ("/"). It supports parsing both simple unit terms and more complex unit expressions with a numerator and an optional denominator. The resulting unit reflects the relationship described by the parsed expression.
