CompositeUnits

com.phasmidsoftware.number.dimensions.core.CompositeUnits
object CompositeUnits

Commonly used composite units defined for convenience. Users can also create their own composite units using the * and / operators.

Note: These are provided as alternatives to the predefined units like MeterPerSecond, Newton, etc. The advantage of composite units is that they can be created on-the-fly for any combination.

For definitions of Imperial Units, see https://www.legislation.gov.uk/uksi/1995/1804/schedule/made
