NumberIsAdditiveCommutativeMonoid

com.phasmidsoftware.number.algebra.eager.Number.NumberIsAdditiveCommutativeMonoid
object NumberIsAdditiveCommutativeMonoid extends AdditiveCommutativeMonoid[Number]

Provides an implicit additive commutative monoid implementation for the Number type.

This object defines the zero element and the plus operation for the Number type, conforming to the algebraic structure of an additive commutative monoid.

The zero element acts as the additive identity, and the plus operation is both associative and commutative, enabling addition of two Number instances.
