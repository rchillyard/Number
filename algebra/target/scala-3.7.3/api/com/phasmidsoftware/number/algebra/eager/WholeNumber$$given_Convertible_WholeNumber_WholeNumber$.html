object given_Convertible_WholeNumber_WholeNumber extends Convertible[WholeNumber, WholeNumber]

Provides a Convertible implementation for converting instances of WholeNumber to WholeNumber.

This implementation allows a WholeNumber instance to be converted to another WholeNumber without applying any transformation, effectively serving as an identity conversion.
