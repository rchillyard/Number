NatZero

com.phasmidsoftware.number.algebra.eager.NatZero
case object NatZero extends Nat

Represents the natural number 0 in Peano arithmetic.

NatZero is the base case of the Nat type, signifying the absence of any successors. It serves as the foundation for constructing all other natural numbers through the Succ type.
