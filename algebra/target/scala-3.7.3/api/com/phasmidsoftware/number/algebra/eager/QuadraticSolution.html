QuadraticSolution

com.phasmidsoftware.number.algebra.eager.QuadraticSolution
See theQuadraticSolution companion object
case class QuadraticSolution(base: Structure, offset: Structure, coefficient: Int, imaginary: Boolean)(maybeName: Option[String]) extends Algebraic

Represents a solution to a quadratic equation in the context of a specific Factor.

Value parameters

base

The base Value component of the solution.

offset

The offset Value component of the solution, usually derived from the discriminant. QuadraticSolution models the structure of solutions to equations of the form: x^2 + px + q = 0. It encapsulates components necessary for representing and interpreting the result in terms of its base, the offset typically arising from the quadratic formula's discriminant, and any contextual modifications by a factor. Instances of this class are designed to interact with the Factor type in performing various operations such as evaluating, converting, or rendering the solution accurately in diverse contexts.
