Radians

com.phasmidsoftware.number.algebra.eager.Radians
See theRadians companion object
trait Radians extends Scalar, Functional

The Radians trait represents a scalar quantity expressed in radians, a unit of angular measure. It extends the Scalar trait, inheriting its properties and behaviors for numerical operations and comparison, while specifically associating the scalar with a conversion factor defined by Pi.

def isExact: Boolean

Determines whether this object is exact, i.e., is not any kind of approximation.

Determines whether this object is exact, i.e., is not any kind of approximation.

CONSIDER it may be possible that there are non-approximatable entities that are not exact either.

The method returns true if the value is represented exactly in IEEE 754 format, or Two's complement or the exact value must be represented symbolically. Examples of the first type include the number one (represented here as either WholeNumber(1) or Succ(NatZero). When we are forced to approximate a value, for example when we want to print the decimal value of , the Golden Ratio, then we mark such a representation as approximate, i.e., this method returns false.
