Eager

com.phasmidsoftware.number.algebra.eager.Eager
See theEager companion trait
object Eager

The Eager object provides factory methods to create instances of Valuable entities that are evaluated eagerly. These entities can represent numerical values parsed from strings, long integers, or specific types of mathematical fields.
