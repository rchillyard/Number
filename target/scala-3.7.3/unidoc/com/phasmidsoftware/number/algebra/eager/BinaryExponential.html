case class BinaryExponential(exponent: Number)(maybeName: Option[String]) extends Exponential

Represents a binary exponential number structure, extending the Exponential class with operations and transformations specifically relevant to base-2 exponential numbers.

This class provides methods to manipulate and interact with the exponential representation, including rendering, comparison, construction of new instances, and providing structural properties.

the exponent value for the binary exponential
