# Number - Scala Mathematical Library

Comprehensive Scala 3 library providing exact and fuzzy arithmetic with sophisticated type hierarchies and lazy evaluation.

## Documentation

**[View Unified API Documentation](api/)**

## About

Number provides:
- Exact arithmetic for rational and algebraic numbers
- Fuzzy arithmetic with uncertainty propagation
- Lazy evaluation and symbolic computation
- LaTeX expression parsing
- Integration with Cats ecosystem
