Root

com.phasmidsoftware.number.expression.expr.Root
See theRoot companion trait
object Root

The Root object provides predefined mathematical constants or solutions to well-known equations modeled as roots. These constants are instances of QuadraticRoot or LinearRoot corresponding to specific root solutions of their respective equations.
