LinearEquation

com.phasmidsoftware.number.expression.algebraic.LinearEquation
case class LinearEquation(r: Rational) extends Equation

Represents a linear equation of the form x + r = 0 where r is a rational number. The slope of the line representing the equation is one, therefore the intercepts are: x = -r when y = 0 (the solution/root); and y = r when x = 0. This class provides various operations and transformations specific to linear equations.
